Artikel HTML

HTML adalah bahasa markup atau penanda untuk membuat sebuah halaman web. Penanda atau markup ini, dikenal dengan istilah Tag. HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah web. Dalam membuat sebuah halaman web, HTML tidak sendirian. Sebagai pelengkapnya HTML dapat dikombinasikan dengan bahasa lain seperti CSS, dan juga Javascript. CSS yaitu bahasa khusus yang digunakan untuk memperindah tampilan web. Sedangkan Javascript bertugas untuk membuat halaman web agar menjadi lebih hidup. Karena dengan Javascript, kita dapat menentukan fungsi - fungsi maupun efek yang akan diterapkan di website. Pada awal kemunculannya HTML mulai diperkenalkan pada tahun 1980, pada saat itu seorang ilmuan bernama Tim Berners-Lee sedang bekerja di CERN yaitu sebuah perusahaan yang berkaitan dengan teknologi internet. CERN berasal dari bahasa perancis "Conseil Europeen pour la Recherche Nucleaire" yang artinnya Komisi Eropa untuk Penelitian Fisika Nuklir. Adapun peralatan untuk belajar HTML yang perlu disiapkan antara lain:

  • Teks Editor untuk menulis kode HTML.
  • Web Browser untuk menampilkan halaman HTML.
  • Direktori atau folder tempat menyimpan berkas (file) yang akan kita buat.
Langkah pertama yang yaitu dengan membuka teks editor, kemudian tulis kodenya, setelah itu simpan dengan nama misalkan hello word.html. apabila menggunakan Notepad di wondows simpan dengan nama filenya menggunakan tanda petik supaya ekstensinya .html bukan .txt atau dapat juga aktifkan fitur show extention pada windows explorer agar bisa mengetahui ekstensi filenya. Apabila sudah langsung saja buka halaman html dengan membuka nama file dengan web browser, maka hasilnya akan terlihat di dalam web browser anda.

Artikel CSS

CSS (Cascade Style Sheet) adalah sebuah bahasa untuk mengatur tampilan web sehingga terlihat lebih menarik dan indah. Dengan CSS, kita bisa mengatur layout (tata letak), warna font, garis, dan lain-lain. CSS pertama kali diperkenalkan oleh Hakon Wium Lie pada tahun 1994. Dari awal diperkenalkan CSS mempunyai beberapa variasi : CSS1 yaitu versi pertama (17 Desember 1996), CSS2 yaitu vers kedua (Mei 1998), CSS 2.1 (7 Juni 2011), CSS3 (2012), CSS4 (masih dalam pengembangan). Struktur kode CSS terdiri dari tiga bagian: Selektor; Blok Deklarasi; Properti dan nilanya. Selektor Selektor adalah kata kunci untuk memilih elemen HTML yang akan kita atur. Artinya: Kita memilih semua elemen h1, lalu diberikan warna teks red (merah). Selektor dapat berupa nama tag, class, id, dan atribut. Pembahasan lebih dalam tentang selektor dapat dibaca pada: Memahami Selektor pada CSS. Blok deklarasi adalah tempat kita menuliskan atribut-atibut CSS yang akan diberikan ke pada selektor. Artinya, kita akan mengatur ukuran font dari tag p sebesar 18px. Blok deklarasi dimulai atau dibuka dengan tanda kurung { lalu ditutup dengan }. Properti dan Nilainya Properti merupakan atribut atau sekumpulan aturan yang akan diberikan kepada elemen yang dipilih. Setiap properti harus diakhiri dengan titik koma (;). Apabila hanya terdapat satu properti, boleh tidak menggunakan titik koma. Properti harus ditulis di dalam blok deklarasi. Tips Belajar CSS. Ada beberapa tips yang ingin saya bagikan untuk membantu teman-teman dalam belajar CSS: Tips 1: Cara Menghapal Kode dan Properti CSS. Ada ratusan properti CSS yang harus diingat. Jika anda sering mengetik kode CSS tanpa copas, bisa jadi ingatan anda terhadap kode atau properti CSS akan semkkin kuat. Tips 2: Gunakan Inspect Elemen. Inspect elemen ada di setiap browser. Bisa dibuka dengan Klik kanan, lalu pilih Inspect Element. Di inspect elemen, kita bisa menulis kode CSS secara langsung dan hasilnya pun akan langsung kita ketahui.

Artikel Javascript

Javascript adalah bahasa pemrograman yang pada awalnya didesain untuk berjalan di atas browser. Namun, seiring berkembangnya zaman, javascript tidak hanya berjalan di atas browser saja. Javascript juga bisa digunakan pada sisi Server, Game, IoT, Dekstop, dan lain sebagainnya. Pada mulanya javascript bernama Mocha, kemudian berubah menjadi LiveScript ketika browser Netscape Navigator 2.0 rilis versi beta (September 1995). Namun setelah itu dinamai ulang menjadi javascript. Terinspirasi melalui kesuksesan javascript, Microsoft mengadopsi teknologi serupa. Microsoft membuat 'Javascript' versi mereka tersendiri bernama JScript. Kemudian ditanam pada internet Explorer 3.0 Hal ini mengakibatkan 'perang browser' karena JScript mlik Microsoft berbeda dengan Javascript racikan Netscape. Akhirnya pada tahun 1996, Netscape mengirimkan standarisasi ECMA-262 ke Ecma Internatinoal, sehingga lahirlah standarisasi kode javascript yang bernama ECMAScript atau ES. Saat ini ECMAScript sudah mencapa versi 8 (ES8). Untuk peralatan yang harus dipersiapkan saat belajar javascript yaitu Web Browser, dan Teks Editor dengan rekomendasi gunakan VS Code. Kemudian untuk penulisan kode di javascript di dalam Html ada beberapa cara yang wajib kita ketahui, yaitu Embeded (kode javascript ditempel langsung pada Html), Inline (kode javascript ditulis pada atribut Html), dan Eksternal (kode javascript ditulis terpisah dengan file Html)
